Please select a paper tape image file to load. If you don't already have one, you can assemble a program to generate a tape image.

An example tape image, hello.tape, is also included in the git repository for the simulator. Download it here.

Here are some getting-started instructions.

  1. Press "Mount Paper Tape" and select the hello.tape file.
  2. Press the CODABO (TSR) button
  3. Press the "Sync: Start" button.
Presently you should see some output.

You can find out more about the simulator project on our website or take a look at the source code.

Machine status information will eventually go here.
